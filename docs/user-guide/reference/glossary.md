# Glossary

Short definitions for common Xtream terms.

## Audio Source

An audio file or extracted audio record in the media pool.

## Blackout

A live safety action that fades or cuts display output to black for the current session.

## Cue

A planned show action. In Stream, cues are represented as sub-cues inside scenes.

## Display

An output window that can be placed on a monitor, projector, or preview screen.

## Display Zone

A target area inside a display. Single displays have one zone. Split displays have left and right zones.

## Flow

The Stream mode for spatial scene authoring with scene cards and relationships.

## Gantt

The Stream mode for reviewing scene timing, cue spans, timelines, loops, and parallel branches.

## Linked Media

Media that stays at its original disk path outside the project folder.

## Copied Media

Media copied into the project folder, usually under `assets/audio` or `assets/visuals`.

## Embedded Audio

Audio contained inside a video file.

## Extracted Audio

Embedded audio that Xtream has extracted into a project audio source.

## Output Bus

A virtual audio output that collects routed sources and sends them to a physical output.

## Patch

The workspace for standalone static aesthetic scenes, simple routed playback, media import, display mapping, virtual outputs, meters, and direct transport.

## Scene

A programmed moment in Stream. A scene can contain audio, visual, and control sub-cues.

## Stream

The workspace and model for scene-based show programming, triggers, sub-cues, manual boundaries, loops, and thread-based playback.

## Sub-Cue

An audio, visual, or control action inside a Stream scene.

## Thread

An independent running branch of Stream playback.

## Timeline

A visible or calculated sequence of scene playback. Stream can show a main timeline and parallel or side timelines.

## Visual Mingle

Display composition behavior that controls how multiple visual layers combine.

## Visual Preview Lane

The visual sub-cue editor lane used to preview, seek, trim, review fades, and place freeze markers.
